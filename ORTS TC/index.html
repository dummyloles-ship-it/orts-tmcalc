Quick Guide

  • Load Data: Click "Load JSON" to import a pathchartdata.js or JSON file.
  • Stations: Multiple markers are averaged. Click 📊 to see details.
  • Speed Limits: Boards (type 3/4) are used automatically. Set "Default speed" to override.
  • Timing: Enter either "Arrival at Source" or "Arrival at Destination" – the other is computed. Both empty = starts at 00:00.
  • Halts: Toggle "Halt" to enable/disable dwell at each station. Unchecked = station skipped.
  • Passing times: When "Use Passing times" is ON, skipped stations show Pass at HH:MM and XXPXX in ORTE format.
  • Buffer: ⬑ icon indicates buffer before arriving at that station.
  • Driver Performance: 1.0 = perfect driver. 0.5 = half speed = double time. 0.0 = stopped.
  • Multi‑edit: Select rows with "Multi-edit" checkboxes, then apply Halt or Buffer.
  • Main Toggles: "Auto‑calc" for real‑time updates. "Auto‑hide columns" hides columns when window is narrow.
  • Column Selection: Toggle any column visibility – Preview & Copy only show visible columns.
  • Preview: "👁️ Preview" shows full table with headers in TSV. "🚆 OR TE" offers two formats.
  • Import: "📥 Import" pastes timings from Preview, Stations+Timings, or Timings Only formats.
  • Settings (⚙️): Theme, defaults, halts enabled by default, auto‑calc, auto‑hide columns, passing times, folder reference.

💡 Hover over any element for a tooltip.
