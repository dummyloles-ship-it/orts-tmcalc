🚆 ORTS Timetable Mode Schedule Calculator – Quick Guide

  • Load Data: Click "Load JSON" to import a pathchartdata.js or JSON file.
  • Stations: Multiple markers for a station are automatically averaged. Click the 📊 button on any row to see the averaging details.
  • Speed Limits: Numeric boards (type 3/4) are used automatically. Set "Default speed" to override all limits (leave empty to use data).
  • Timing: Enter either "Arrival at Source" or "Arrival at Destination" – the other is computed. Both empty = starts at 00:00.
  • Halts: Toggle the "Halt" checkbox to enable/disable dwell time at each station. Unchecked = halt forced to 0.
  • Buffer: The ⬅ icon indicates buffer before arriving at that station (between previous → current).
  • Multi‑edit: Select rows with the "Apply edits for" checkbox, then choose "Apply to all" for Halt or Buffer.
  • Column Selection: Toggle any column visibility – Preview & Copy will only show visible columns.
  • Preview: "👁️ Preview" shows the full table (with headers) in TSV format. "🚆 ORTS Preview" offers two formats: with or without station names.
  • Settings (⚙️): Change theme, default halt, default buffer, or save a folder reference. Theme changes preview instantly – click "Save Settings" to keep them.
  • Themes: Dark (default), Light, Solarized, Monokai, Ocean – choose your preference.
