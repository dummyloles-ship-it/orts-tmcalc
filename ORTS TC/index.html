🚆 ORTS Timetable Mode Schedule Calculator – Quick Guide

  • Load Data: Click "Load JSON" to import a pathchartdata.js or JSON file.
  • Stations: Multiple markers for a station are automatically averaged. Click the 📊 button on any row to see the averaging details.
  • Speed Limits: Numeric boards (type 3/4) are used automatically. Set "Default speed" to override all limits (leave empty to use data).
  • Timing: Enter either "Arrival at Source" or "Arrival at Destination" – the other is computed. Both empty = starts at 00:00.
  • Halts: Toggle the "Halt" checkbox to enable/disable dwell time at each station. Unchecked = halt forced to 0 and station is skipped (no arrival/departure shown).
  • Buffer: The ⬑ icon indicates buffer before arriving at that station (between previous → current). Buffer is added on top of running time.
  • Driver Performance: A factor from 0.0 to 1.0. 1.0 = perfect driver (theoretical running time). 0.5 = half speed = double running time. 0.0 = train stopped. Buffer and halt are added on top.
  • Multi‑edit: Select rows with the "Multi-edit" checkbox, then choose "Apply to all" for Halt or Buffer.
  • Column Selection: Toggle any column visibility – Preview & Copy will only show visible columns.
  • Preview: "👁️ Preview" shows the full table (with headers) in TSV format. "🚆 OR TE Format" offers two formats: with or without station names – you can edit the text before copying.
  • Settings (⚙️): Change theme, default halt, default buffer, enable/disable halts by default, toggle automatic calculation, or save a folder reference. Theme changes preview instantly – click "Save Settings" to keep them.
  • Themes: Dark (default), Light, Solarized, Monokai, Ocean – choose your preference.
